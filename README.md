**Deskripsi Aplikasi**
VOIZ-FTMM adalah sistem e-complaint yang dirancang untuk memfasilitasi pengaduan atau komplain kepada Fakultas Teknologi Maju dan Multidisiplin (FTMM). Aplikasi ini dapat digunakan oleh mahasiswa, tenaga kependidikan (tendik), dosen, seluruh civitas akademika FTMM, serta pelapor anonim.
